Q: Explain how to calculate width and height of an element

(Element: Height = 320px, Width = 550px; Content Area: Height = 311, Width = 542)

  • The "Height" and "Width" properties do not add borders, margins or padding, however, it udjusts the height and width of the content within them.
  • The "Height" and "Width" properties can have multiple values, including:
    1. Auto: sets it to automatic, and lets the browser decide the Height/Width. This is the default setting.
    2. Length: lets you set the Height/Width through defining things like centimeters, pixels, e.t.c.
    3. %: adjusts the Height/Width based on a percentage.
    4. initial: resests the Height/Width to its default value.
    5. inherit: the parent value of Height/Width will be used.

    • Q: Explain how to calculate width and height of an element, using the box-model

    (Element: Height = 259, Width = 551; Content Area: Height = 250, Width = 543)

  • The "box-model" in CSS refers to designs and layout. The box-model is a 'box' that wraps around the elements in a HTML page. it consists on Margins, Borders, Padding, and the Content.
  • In order to know how Height/Width works in HTML and CSS, it is important to know how the 'box-model' works. When we set the Width in in CSS it only applies to the content area, but, if we want it to apply tot he entile HTML element, we must add padding, borders and margins also.
