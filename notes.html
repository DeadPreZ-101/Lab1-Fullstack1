Q: Explain how to calculate width and height of an element

(Element: Height = 320px, Width = 550px; Content Area: Height = 311, Width = 542)

  1. The "Height" and "Width" properties do not add borders, margins or padding, however, it udjusts the height and width of the content within them.
  2. The "Height" and "Width" properties can have multiple values, including:
    • Auto: sets it to automatic, and lets the browser decide the Height/Width. This is the default setting.
    • Length: lets you set the Height/Width through defining things like centimeters, pixels, e.t.c.
    • %: adjusts the Height/Width based on a percentage.
    • initial: resests the Height/Width to its default value.
    • inherit: the parent value of Height/Width will be used.

Q: Explain how to calculate width and height of an element, using the box-model

(Element: Height = 259, Width = 551; Content Area: Height = 250, Width = 543)
  1. The "box-model" in CSS refers to designs and layout. The box-model is a 'box' that wraps around the elements in a HTML page. it consists on Margins, Borders, Padding, and the Content.
  2. In order to know how Height/Width works in HTML and CSS, it is important to know how the 'box-model' works. When we set the Width in in CSS it only applies to the content area, but, if we want it to apply tot he entile HTML element, we must add padding, borders and margins also.

The Position Attribute

  1. What does position absolute do?
    • Absolute positioning means the element will be relative to its nearest 'ancestor'. If there is no ancestor, then it will use the page's body and move with scrolling.
  2. What does position realtive do?
    • When this is used - the element in question's position changes to become relative to its default/normal position. We can set attributes like top, right, bottom and left to chnage the postion to reflect new position away from its usual place.
  3. What does position fixed do?
    • Elements which are 'fixed' makes those elements always stay in the smae place, even if the page is scrolled. There are top, right, bottom, and left options that can be applied to move the element around.
